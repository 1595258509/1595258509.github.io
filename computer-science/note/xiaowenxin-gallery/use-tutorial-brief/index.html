小温新图库网址：10.10.8.24
仅可在学校内网中访问（宿舍宽带、i-WZU，WZU-free...）

# 注册账号

点击注册账号注册

按照要求输入信息 注意取消选择下面的选框

# 上传照片

须创建账号并登录才可进行照片上传

登录后，选择顶栏中的上传图片

# 创建相册后上传

点击创建新相册，父相册按 本次所拍摄的主题 进行选择，并创建以 拍摄人姓名 为相册名的子相册

之后就可以进行照片上传啦

# 已有相册上传

若补传图片只需要选中之前自己创建的相册即可。

相册路径规范 ： 年份 + 拍摄主题 + 拍摄人

# 下载照片

须登录账号方可下载

先进入到需要下载的相册当中

点击图中的按钮即可下载
